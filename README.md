# ros2_ws
Collection of packages of ROS packages for Gazebo worlds, robotic arms and autonomous cars.

Run the simulation using `colcon build && source ~/.bashrc && ros2 launch table_tennis_gazebo simulation.launch.py` and `source ~/.bashrc && ros2 launch table_tennis_gazebo ball_spawn.launch.py x:=0.3 y:=0.5 z:=2.0`
